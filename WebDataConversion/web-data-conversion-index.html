🗂 Web Data Conversion Lab シリーズ マスター索引

ブラウザとWebサーバー間で行われる全データ変換を、JavaScriptで再現・体感する全7 Labのロードマップです。
「通信のどこで起きるか」をレイヤーで整理し、送信↔受信の双方向を網羅しています。

🎬 「大阪 ラーメン」で検索すると、裏側で7回データが姿を変える

ブラウザの検索バーに「大阪 ラーメン」と入力して検索ボタンを押す。
たったその1つの操作で、データは7回も姿を変えてサーバーに届きます。

① 入力: "大阪 ラーメン" ↓ URLエンコード（Lab 01） ② "%E5%A4%A7%E9%98%AA+%E3%83%A9%E3%83%BC%E3%83%A1%E3%83%B3" ↓ HTTPリクエスト構築（Lab 05） ③ GET /search?q=%E5%A4%A7%E9%98%AA... + Content-Type + Cookie ↓ UTF-8バイト列化（Lab 03） ④ [0x47, 0x45, 0x54, 0x20, ...] ↓ TLS暗号化（Lab 06） ⑤ 解読不能な暗号文 ↓ TCP/IPパケット化 ⑥ 電気信号として送信 ↓ サーバーで逆変換（Lab 07） ⑦ "大阪 ラーメン" ← 元に戻った！

このLabシリーズでは、上記の各ステップを自分の手で再現・確認できます。Lab 01から順に進めましょう。

🚀 Quick Start

1. サーバー起動

node -e "const http=require('http'),fs=require('fs'),path=require('path');http.createServer((req,res)=>{const f=path.join('C:/Users/user/ClaudeCode/WebDataConversion',req.url==='/'?'/web-data-conversion-index.html':req.url);fs.readFile(f,(e,d)=>{if(e){res.writeHead(404);res.end('Not found');}else{const ext=path.extname(f);const mime={'html':'text/html','js':'application/javascript','css':'text/css','png':'image/png','ico':'image/x-icon'};res.writeHead(200,{'Content-Type':(mime[ext.slice(1)]||'text/plain')+';charset=utf-8'});res.end(d);}});}).listen(7891,()=>console.log('Server: http://127.0.0.1:7891'));"

サーバーが http://127.0.0.1:7891 で起動します。

2. ブラウザでアクセス

http://127.0.0.1:7891/web-data-conversion-lab-XX.html

XX を Lab 番号（01〜07）に置き換えてください。

この索引ページ自体も同じサーバーで配信できます: http://127.0.0.1:7891/
HTTP Header Lab（ポート7890）と同時に起動可能です。
