// このアプリの使い方

このアプリは 読み取り専用 のフラッシュカードプレイヤーです。 カードはGoogle Driveの Flashcards/ フォルダ内のJSONファイルから読み込まれます。

// カードを追加する

  1. Claude(または手動)でフラッシュカード用JSONを作成
    Claudeのスキル「flashcard-json-builder」に「カード化して」と依頼すれば自動生成されます
  2. 生成された flashcard-import-YYYY-MM-DD.json 等を、 Driveの マイドライブ > Flashcards フォルダにアップロード
  3. アプリで「⬇ Driveから再読込」をクリック → 即座に反映

// カードを編集・削除する

  1. Drive上の対象JSONファイルを直接編集(またはClaudeに依頼して再生成)
  2. 「Driveから再読込」で反映
📘 設計思想: カードは「教材」、レビュー履歴は「学習記録」として明確に分離しています。 カードはClaude+スキルで管理し、アプリは想起練習に専念する構造です。

// 学習履歴の保存場所

自己評価(○△×)とレビュー回数は、Driveの Flashcards/_reviews.json に 自動的に書き戻されます。複数端末で同じアカウントでサインインすれば、学習履歴は同期されます。

// セットアップ手順

  1. Google Cloud Console にアクセス
  2. 新規プロジェクトを作成(例: flashcards)
  3. 「APIとサービス」→「ライブラリ」→ Google Drive API を有効化
  4. 「APIとサービス」→「OAuth同意画面」→ ユーザータイプ「外部」→ アプリ名・サポートメール等を入力
  5. スコープ追加: 以下の2つを追加
    • ../auth/drive.readonly — Drive内のJSONファイル読込用
    • ../auth/drive.file — _reviews.json作成・更新用
  6. テストユーザーに自分のメールを追加
  7. 「認証情報」→「認証情報を作成」→「OAuth クライアントID」→ 種類「ウェブアプリケーション」
  8. 承認済みのJavaScript生成元に、このアプリの公開URL(例: https://username.github.io)を追加
  9. 表示されたクライアントIDを上の欄にコピペして「保存して再読込」
