🚀 Quick Start

1. サーバー起動

node server.js

サーバーが http://127.0.0.1:7890 で起動します。

2. ブラウザでアクセス

http://127.0.0.1:7890/http-header-lab-XX.html

XX を Lab 番号（01〜16）に置き換えてください。

この索引ページ自体も同じサーバーで配信できます: http://127.0.0.1:7890/http-header-lab-index.html
