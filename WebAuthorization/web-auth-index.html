🗂 Web認証 Lab シリーズ マスター索引

Web認証・認可の仕組みを、ブラウザから実際にリクエストを送って体験する全3 Auth Labのロードマップです。

🔰 初心者向け: 認証とは「あなたが誰か」を確認する仕組みです。 Webサイトにログインするとき、IDとパスワードを入力しますよね。 その裏側でブラウザとサーバーがどんなデータをやり取りしているかを、このLabシリーズで体験します。

🚀 Quick Start

1. サーバー起動

node server.js

サーバーが http://127.0.0.1:7892 で起動します。

2. ブラウザでアクセス

http://127.0.0.1:7892/web-auth-lab-XX.html

XX を Lab 番号（01〜03）に置き換えてください。

配信フォルダ: C:/Users/user/ClaudeCode/WebAuthorization
